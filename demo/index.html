Voice-Driven Scrolling

Speech recognition follows your pace. Text scrolls as you speak.

🖱

Click-Through

Markdown

Headers, bold, italic, lists, code blocks. Write in markdown, present beautifully.

Always On Top

Install

Compiles from source on your Mac. No signing, no Gatekeeper.

With Claude Code

/plugin update gusto-gists
/plugin install teleprompter@gusto-gists

# Then run:
/teleprompter

Manual

Requires Xcode Command Line Tools

# Clone and compile
git clone https://github.com/lukeatgusto/teleprompter.git /tmp/tp
swiftc /tmp/tp/teleprompter.swift -o ~/.local/bin/teleprompter \
  -framework Cocoa -framework WebKit \
  -framework Speech -framework AVFoundation
rm -rf /tmp/tp

# Run
~/.local/bin/teleprompter
Best Practices
1

Position near your camera

Overlap the teleprompter with your webcam to maintain eye contact while reading.

2

Tune opacity to your background

Light desktop? Lower opacity. Dark? Raise it. Readable text, visible background.

3

Use voice mode for presentations

The mic follows your pace. No fiddling with scroll speed.

4

Dark mode for light backgrounds

Hit the moon icon for black text when overlaying bright slides.
