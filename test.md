# Welcome to Teleprompter

This is a test of the **teleprompter overlay**. You should be able to see through the dark background to whatever is behind this window.

## Key Points

- The background is *translucent*, not opaque
- Text is large and easy to read
- Auto-scroll keeps your place

### How to Use

1. Press **Space** to start scrolling
2. Use **Up/Down arrows** to adjust speed
3. Press **R** to reset to the top
4. Press **[** and **]** to adjust font size
5. Press **-** and **=** to adjust opacity
6. Press **M** for mirror mode
7. Press **Q** or **Esc** to quit
